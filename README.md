# Data Mining and Knowledge Discovery (GVSU – CIS 635)

Welcome to the **Data Mining and Knowledge Discovery** course repository for *Grand Valley State University*. This
repository supports CIS 635 and contains **lecture materials, notebooks, datasets, assignments, projects, and
presentations** that guide students through the theory and practice of modern data science.

---

## 📌 Course Overview

In today’s world, data is being generated at an unprecedented scale—**measured in zettabytes**. In fact, the total
amount of data produced before 2010 is estimated to be **less than 0.1 zettabyte**, a fraction of what is generated in a
single year today. As this massive wave of data continues to grow in both **volume and velocity**, the need to
systematically extract useful knowledge from data has become critically important.

**Data Science** has therefore emerged as a key discipline tasked with answering how best to **analyze, interpret, and
utilize large-scale data**. This course introduces students to **computational methods for knowledge discovery and data
mining**, emphasizing practical, data-driven problem solving.

Topics covered include **Python programming (specialized for data science)**, **data preparation**, **data
visualization**, **predictive modeling**, **model evaluation**, **clustering**, and **association analysis techniques**.
The course emphasizes both **supervised and unsupervised learning methods** and the effective communication of insights
derived from data.

---

## 🎯 Course Objectives

After completing this course, students should be able to:

1. **Wrangle, analyze, and visualize data** using Python and modern data science libraries.
2. **Exercise and apply supervised and unsupervised** knowledge discovery techniques.
3. **Identify and build models** using common data mining and predictive modeling approaches.
4. **Evaluate models** for their effectiveness, validity, and appropriateness.
5. **Communicate findings effectively** using clear, informative visualizations and written analysis.

---

## 🧠 Topics Covered

Throughout the course, students will engage with:

- **Python for Data Science:** NumPy, pandas, matplotlib, seaborn
- **Data Preparation:** Cleaning, transformation, feature engineering
- **Exploratory Data Analysis (EDA):** Statistical summaries and visualization
- **Predictive Modeling:** Classification and regression techniques
- **Model Evaluation:** Validation strategies and performance metrics
- **Unsupervised Learning:** Clustering methods (e.g., k-means, hierarchical)
- **Association Analysis:** Frequent itemset mining and rule discovery
- **Knowledge Discovery Process:** From raw data to actionable insight
- **Ethical Considerations:** Responsible and reproducible data analysis
